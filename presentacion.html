1 CONOCER MÁS DE HTML

Se encarga de desarrollar una descripción sobre los contenidos (textos, fotografías, animaciones, etc)...
Al acceder mediante cualquier navegador web, lo que hace el navegador, es pedirle al servidor que le envíe dicho documento, entonces el servidor le enviará dicho documento HTML donde se encuentra el código HTML, el navegador recibe el documento, lo lee, interpreta y muestra el contenido al usuario final...
Por ejemplo, como se muestra la imagen de una página web de una tienda de ropas.


2 CONOCER MÁS DE CSS

Es un mecanismo complementario del lenguaje HTML que permite indicarle al navegador el estilo que debe darle a los distintos elementos al desplegar la información de un sitio web...
Por ejemplo, como se puede observar en la imagen de una encuesta sobre el coronavirus (COVID-19).


3 CONOCER MÁS DE JAVASCRIPT

Se utiliza principalmente del lado del cliente (es decir, se ejecuta en nuestro ordenador, no en el servidor) permitiendo crear efectos atractivos y dinámicos en las páginas web. Los navegadores modernos interpretan el código JavaScript integrado en las páginas web.
Por ejemplo, la imagen es un filtrado de ropas para niños.


4 CONOCER MÁS DE PHP

Es un lenguaje de programación interpretado que se utiliza para la generación de páginas web de forma dinámica. Éste código se ejecuta al lado del servidor y se incrusta dentro del código HTML. Cabe destacar que es un lenguaje de código abierto, gratuito y multiplataforma.
Por ejemplo, observamos en la imagen un registro de usuario en la que está conectado con una base de datos, por ende, al rellenar la informacióon se guardará.
